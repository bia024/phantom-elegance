# Planner do Projeto

## Visão Geral
Breve descrição: Planejador simples para organizar tarefas do projeto.

## Como usar
- Atualize a seção "Diário" ao começar o dia.
- Mova itens do "Backlog" para "Em Progresso" quando iniciar.
- Use prioridades e prazos para ordenar tarefas.

## Seções
### Diário
- Data: ____
- Objetivos do dia:
  - [ ] Tarefa 1
  - [ ] Tarefa 2

### Semanal
- Semana: ____
- Objetivos principais:
  - [ ] Entregável A
  - [ ] Revisão B

### Backlog
- Ideias e tarefas futuras:
  - [ ] Feature: descrição curta
  - [ ] Bug: descrição curta

### Kanban (colunas sugeridas)
- Backlog
- Para Fazer
- Em Progresso
- Revisão
- Concluído

### Modelos de tarefa
- Título: _Breve título_
- Descrição: _O que precisa ser feito_
- Prioridade: Alta / Média / Baixa
- Prazo: YYYY-MM-DD
- Responsável: _Nome_

## Exemplo de tarefas iniciais
- [ ] Ajustar layout do componente `Header` (Prioridade: Alta, Prazo: 2025-12-20)
- [ ] Criar galeria responsiva (Prioridade: Média)
- [ ] Revisar README e documentar o planner (Prioridade: Baixa)
